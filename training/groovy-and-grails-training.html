Duration: 5 days

Course Overview:

This training course is meant to introduce Groovy and Grails to software engineers with experience in Java Enterprise development. Students will get a thorough introduction of the technology and will be walked through aspects of the Groovy programming language and the Grails web application framework.

This course will be taught by instructors who are among the Philippines' seasoned software developers. O&B has solid experience with Grails - with a portfolio that includes developing robust web applications - many processing millions of mission-critical transactions a day.

Prerequisite:

Core Java & Commonly-Used Libraries, Servlets/JSP, JDBC, SQL

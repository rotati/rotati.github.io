  • All Angular JS Courses
  • Angular JS Course (Beginner)

Duration: 2.5 days

Course Overview:

Angular JS Course for Beginners is aimed at developers who already have some understanding of HTML and JavaScript and are interested to take their web application development skills to the next level. Candidates for this course have heard about the power of Angular JS and are excited to learn more about it and gain some real experience!

Our experienced, international teachers will guide students through the journey of getting setup and running with their first Angular JS application. Students will learn about all the features of Angular JS such as two-way data binding, dependency injection, creating reusable components using directives and much, much more.

Prerequisite:

Students must already have some programming background, such as through school, basic training courses, or work experience - particularly with basic HTML and JavaScript.

Course Outline:

  • Introduction to AngularJS
  • Hello World in Angular JS
  • $compile, $digest and $apply Model Views
  • Dependency Injection
  • Modules
  • Module Constants
  • Module Run
  • Controllers and $scope
  • Routes with Controllers
  • Views
  • Directives
  • Layout Directives
  • Event Directives
  • Styling Directives
  • Services in AngularJS
  • Build Your First Service
  • The $http Service
  • Communicating with Controllers
  • Directive Definition Object
  • Link Function
  • Controller Function
  • Angular-Translate

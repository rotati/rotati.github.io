Duration: 3 days

Course Overview:

Spring is a dependency injection framework that simplifies Java EE development and testing.

This hands-on training teaches you how to apply Spring to create Web applications.

Rotati is the first SpringSource partner based in the Cambodia. Many of our engineers and architects have been using Spring Framework since its first stable release in 2004, and have built massive global systems as well as innovative products using Spring.

Benefits of the program:

Trainees will learn to do the following:

  • Use the Spring Framework to develop Web applications
  • Implementing RESTful services using Spring MVC
  • Secure Web applications with Spring Security
  • Use Spring Web Flow to implement stateful interactions
  • Building rich clients using AJAX and JavaScript
  • Test Web applications for correctness

Audience:

Experienced enterprise Java developers and architects who want to learn how to take advantage of the Spring Framework in their web applications.

Prerequisite:

  • Experience with developing applications using Java (including Servlet API, JUnit)
  • Basic understanding of Spring (including dependency injection, application configuration)

Course Outline:

  • Getting Started with Spring Web MVC
  • Spring MVC Configuration Options
  • Managing Layouts in Spring MVC
  • Using Views in Spring MVC
  • Building Form Pages with Spring MVC
  • Implementing REST
  • Exception Handling
  • Web Application Security with Spring Security
  • Debugging and Testing Web Applications
  • Building Rich Web Applications with Ajax (optional)
  • Getting Started with Spring Web Flow
  • Spring Web Flow Language Essentials and Actions
  • Spring Boot and Websockets (optional)

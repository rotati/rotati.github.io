Course Overview:

Ionic is an excellend Hybrid Mobile Application framework for building sleek, beautiful Hybrid Mobile applications targeting iOS and Android.

If you already know HTML5, CSS, JavaScript and Angular and you want to learn how to use these technologies with a powful technology like Ionic then this course is for you.

Modern frameworks such as AngularJS and Ionic make it easier to build Mobile Applications than ever! Ionic ships with useful and powerful pre-built templates and components. In this course, students will learn about the core Ionic components: prebuilt, easy-to-use features such as buttons, lists, tabs, and forms. Students will also learn how to combine componets together into Angular JS Models, Views and Controllers and build amazing single page Hybrid Mobile Applications!

Prerequisite:

Students must already have some programming background, such as through school, basic training courses, or work experience - particularly with HTML, CSS, JavaScript and Angular JS. If you need some experience in Angular JS first, then please consider taking our Angular JS Beginner course.

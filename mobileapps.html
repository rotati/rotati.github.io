We love building HTML5 and Hybrid Mobile Apps using Angular JS !

Angular JS is always our first choice, client-side framework for building Mobile Apps. Its well designed, easy to use and a proven technology. Here is a brief explaination of the two types of Mobile Applications that we build here at Rotati!

Get In Touch

HTML5 Applications

HTML5 is a way to make Mobile (and Web) applications run in a browser. By using Angular JS as the application framwork, we can build robust, fast HTML5 mobile apps in a breeze! The application is accessed by tying in a URL into the browser on the mobile phone or finding it via Google. The alternatice is to have a Hybrid / Native mobile app which is downloaded from the various app stores (see column to the right)

Charataristics

  • Application works in the browser.
  • Application is 'loaded' via a url
  • User either knows the url of the application or can search for it in Google.

Hybrid Mobile Applications

Hybrid Mobile Applications are made with the same web technoligies as HTML5 applications (html, css and JavaScript + Angular JS), however they are converted to a (near) native applicaton for iPhone and Android. This allows for deployment of the application to the various mobile app stores.

Charataristics

  • User installs application from app stores
  • Application looks and feels (almost) like a pure native app (i.e. a mobile app build directly using the native technology rather that web technology)

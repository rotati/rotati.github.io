We love building Web Apps and RESTful API's using Ruby on Rails!

These days Ruby on Rails is used to build rock solid API's for providing data to single page JavaScript applications or Mobile Applications. Here is a brief explaination of the two types of Rails Applications that we build here at Rotati!

Get In Touch

Ruby on Rails Applications

Ruby on Rails Applications that are built from the ground up using the Ruby on Rails framework and will contain all the business logic, controlling logic and views for the application. The applications work like traditional web applicatioins in that on each click of a link or a button the entire page is reloaded and served back the user.

Charataristics

  • Application follows the request / response pattern.
  • Application built entirely using the Rails Framework

Ruby on Rails API Applications

API's are 'backend' applications that run on the server and are consumed by 'client' applications such as Single Page JavaScript applications or rich client mobile applications.

Charataristics

  • The Rails API application is consumed by a clinet application
  • API applications separate the concerns of business / data tier and the UI / UX of the client application

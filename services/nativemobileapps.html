We love building Native Mobile Apps using RubyMotion !

Native Mobile Apps
RubyMotion is always our first choice platform for building Native Mobile Apps. Its well designed, easy to use and a proven technology. The thing that we love about RubyMotion is that it allows our seasoned Ruby Developers to build real, nativly running mobile applications using the programming language they know and love, Ruby.

RubyMotiion allows us to use the full Ruby programming API + gems like Rake (for tasks like building etc) and be able to confidentlly test the app using RSpec. Best of all, RubyMotion is a wrapper around the native API's so that we can actually build real native apps in Ruby! RubyMotion provides a familiar environment for the Native Mobile App Developer so that they can produce clean, fast, beautiful and efficient running mobile applications.

Native Game development
Its also possible to build games using RubyMotion using RubyMotion Game that comes included with RubyMotion. Here is an example of FlappyBird in 100 lines of RubyMoton. We can write games using RubyMotion Game for both platforms and a single code base!

Testing Native Applications
By using RubyMotion we have access to RSpec for testing native applications. In addition to having a suite of different Apple and Android hardware devises to hand we can also make use of very powerful, cloud based testing environments to auto test features on 100's of different devises simultaneously. Our favourite cloud based testing environment is Amazon Device Farm.
